# github token

1. google搜索`github token`
2. 创建新token（classic）
3. 设置必填项目
4. 设置`Select scopes`，`repo`项目选择`public_repo`即可。
5. 保存token

# 安装vscode

# 安装插件

插件配置：设置`Github Token`
