# 获取方式

## 自制

无优化，只适合diy

## Codesourcery

它的ARM GCC在目前在市场上非常优秀，但是下载不方便。并且开源版本停更了。

## Linaro（推荐）

ARM等大厂开源支持

# 组件

- Assembler：as
- Binary Utilities：
  - ar
  - nm
  - objcopy
  - objdump
  - ranlib
  - size
  - strings
  - strip
  - c++filt
  - addr2line
  - readelf
  - elfedit
- C Library (Newlib)：libc
- Compiler 
  - gcc
  - g++
  - c++
- Debugger：gdb
- Linker：ld
- Math Library (Newlib)：libm
- Preprocessor：cpp
- Profiler：gprof
